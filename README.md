# KAT Overlap

KAT Overlap — détection et analyse des chevauchements spatiaux (mono / multi-couches) pour QGIS.

**Fonctionnalités clés**
- Analyse intra-couche et inter-couches
- Nettoyage automatique des géométries invalides
- Index spatial pour accélérer les comparaisons
- Rapport interactif + export TXT / XLSX
- Création d'une couche mémoire stylisée des chevauchements
- Support multi-CRS (projection UTM/polaire automatique)

**Installation rapide**
```bash
git clone https://github.com/AzizT-dev/kat_overlap.git
# zipper le dossier kat_overlap puis : QGIS > Extensions > Installer depuis un ZIP...
